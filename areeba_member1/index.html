📖 About IBE Parameters

The Boneh-Franklin IBE scheme uses bilinear pairings over elliptic curves. The KGC generates a master secret key (MSK) and publishes the master public key (MPK). Any sender can encrypt a message to identity ID using only MPK and ID — no per-user certificate required. The KGC uses MSK to extract private key SK_ID = MSK · H₁(ID) for authenticated users.
